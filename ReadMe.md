|-------------------------|-----------------------------|
| GET /files              | List of PDF fileInfo's info |
| POST /files             | Upload PDF fileInfo         |
| GET /files/pdf?fileUID= | Get Specific pdf fileInfo   |
| GET /files/mxl?fileUID  | Get Specific mxl fileInfo   |
